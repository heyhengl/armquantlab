## 中文说明

ArmQuantLab 是一套在 Arm64 本机上复现 ONNX 动态 INT8 量化权衡的证据工具。它使用相同模型、输入与批次配置，对比 FP32 和 dynamic per-channel INT8 的模型体积、延迟与输出相似度，并冻结原始模型、量化模型和报告哈希。

冻结结果显示模型体积减少 74.229%，输出 cosine similarity 高于 0.99989。延迟并非全部改善：batch 8 为 1.281611 倍加速，而 batch 1 与 batch 32 分别只有 0.794915 倍和 0.644923 倍，意味着更慢。项目保留这些负结果，避免把量化描述成无条件性能提升。

公开源码：`https://github.com/heyhengl/armquantlab`

## English description

ArmQuantLab is an evidence-first ONNX quantization workflow for Arm64. It compares FP32 with dynamic per-channel INT8 using the same model, inputs, and batch configuration, then freezes model hashes, benchmark measurements, and parity evidence.

The frozen result reduces model footprint by 74.229% while retaining cosine similarity above 0.99989. Latency is not uniformly better: batch 8 reaches a 1.281611x speedup, while batch 1 and batch 32 are slower at 0.794915x and 0.644923x. The project keeps those regressions visible instead of presenting quantization as an unconditional win.

Public source: `https://github.com/heyhengl/armquantlab`

The visuals are generated by code and the narration uses a synthetic voice. The upload must be declared as AI-generated or synthetic content. No third-party music or human footage is used.
